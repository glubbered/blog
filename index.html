Design is Communication

Creating Software Design should be simple and effortless to achieve its goal, to clarify abstract problems, to communicate with each others and create innovations.

Sketchboard.io is specialised on agile software design. It provides diagramming elements and drawing style that you don't concentrate on details. To get most important out, your idea, not how it looks like.

Realtime Collaboration

Sketchboard.io makes collaboration extremely easy, just open your browser and share the board url. Create ideas together with clients, team members and contractors, in real time.

Work from multiple locations and from different computers, everything is stored in the cloud. You will always have your latest diagrams and sketches with you.

UML, Freehand, Mind Map, Notes and Roadmap

Sketch UML not to forget the big picture. Brainstrom with Mind Map. Draft sketches using freehand. Visualize complex flows. Use Notes to draft daily ideas and highlight objectives using Roadmap.

Mix and match elements on virtual infinite board. Fit all related ideas on one "paper".

All Features
Feature Purpose/Description Free Business Example Usage
Realtime Collaboration Edit board together in realtime. ✔ ✔ Keep online meetings or ad hoc sessions to solve problems as you would sketch on a white board in a same room.
Virtual Endless Board You will never get out of drawing space. Combine multiple sketches on a one board. ✔ ✔ Create multiple sketches on a same board. E.g. sketch freehand user interface + flows from UI actions + highlevel software internal design. You got a map to follow when concentrating details of each topic. Link e.g. from User Story to the board using URL.
Markdown Notes Decorate text on the board. Works for Notes and Just Text elements. ✔ ✔ Write e.g. content of a web page. Just place elements roughly where those belong and concentrate on the content.
Agile UML Agile UML is not strict and allows you to combine any elements with each other. You don't need to be strict on your design. Understanding and communication is the most important thing. ✔ ✔ Combine e.g. server and database with component elements. Sketch some freehand on top of that.
Mind Mapping Idea to provide Mind Mapping elements is to brainstorm ideas. ✔ ✔ Try to e.g. find out what kind of things are related to the problem at hand. Use mind maps to innovate features or services. Mind Maps are good to structure and find related topics.
Continuous Freehand Drawing Enrich your sketch with freehand drawing. ✔ ✔ Create e.g. freehand annotations to highlight certain areas. Use freehand when shape diagramming would be too limiting.
Flow Charts Visualize complex flows. ✔ ✔ With flow chart you can even build site maps. What is the application navigation logic. Clarify complex flows to be more understandable.
Roadmap Highlight Team objectives using Roadmap elements. ✔ ✔ Visual Roadmaps simplifies understanding the goals. It also streamlines different team activities.
Undo/Redo Made a mistake, quickly undo what you did. ✔ ✔ Accidentally deleted some of the items from the board. Press Cmd-Z and you got element back to the board.
Map View Map View shows whole board at once. Press Z key to enable it. ✔ ✔ Map View is especially useful when big board do not fit to the screen. To enable Map View keep z key down. Recenter the board by moving your mouse and release z key. Map View is only enabled when elements are not edited.
Zoom Zoom board using shortcuts, + and - keys. ✔ ✔ Sometimes you want to zoom out to see bigger area of the board at once.
Shows users on Board See other users on the same board. ✔ ✔ Makes easier to see what others are doing. Prevents you to edit same diagram elements.
Chat Board level chat ✔ ✔ Users on the same board can chat with each other.
Https All server communications use HTTPS. ✔ ✔ HTTPS protects data from eavesdropping.
Download Board Download board as PDF, PNG or SVG. ✔ Sometimes it is not enough to collaborate realtime on a board. You might want to have boards on your local machine to work offline. Make e.g. annotations on top of PDF.
Images Upload images on your board. Space is increased by 100 MB per user in the account.
Example:
1 user => 100 MB
2 users => 200 MB
 ✔ Drag and drop e.g. whiteboard drawing, keep meeting minutes as a whole in Sketchboard.io or draw on top of whiteboard drawing.
NOTE: expired account images are removed after 14 days. Read more.
Versioned Boards Create named versions, list versions, view and revert versions.
Versioned boards provides top most protection for your data. Versions are stored on Amazon S3.		 ✔ Versions provide e.g. protection from accidental element deletion or possibility to switch between alternate designs.
NOTE: expired account versioned boards are removed after 14 days.
Read more how versions can be used in your planning meeting or check out potential version usage in sketch design breakdown.
Print Board Quickly print a board directly from the browser. ✔ You don't need to download PDF first and then print.
Change Background Color Possible background options: White, Sepia, Black and Paper. ✔ Sometimes white background is too harsh for your eyes. Especially when you are working in a dark room, black or sepia backgrounds are better for you eyes.
Comments & Annotations Have offline collaboration or raise open items. Comments visibility can be toggled. ✔ Team members are not always using a board at the same time. Sometimes they look sketch at different times. Since sketching is mostly about communication, clarifying the problem for your self or with others, comments makes possible to communicate questions or ideas without using realtime collaboration. Once you are done with comments you can close them. Read More.
Share Boards Securely Share boards securely within your account. All members of an account can view and edit boards.
Note that pricing is based on number of users in the account.		 ✔ You can e.g. add your team to your account and enable secure and easy sharing between team members. Read More.
Private Boards By default board is private and no one else is not able to access the board. ✔ When you want to draft things out by your self without worrying others to see your work. Set board visibility to private. You can toggle board visibility from "Switch Board" or from "Share Board" menu.
Password Protected Boards Share boards using a password, password sharing is not calculated into pricing. ✔ Give e.g. temporary access to clients when you don't want to share all account boards.
Named Boards Give name to your boards. ✔ Name is visible on browser tab as well.
Account Management Manage who can see account level boards. Pricing is based on number of users in the account. ✔ Add your team members to the account and everybody can create and edit boards by them selves or together. All users can have unlimited number of private boards.
Google Drive Integration Add shortcut to Google Drive. ✔ Access boards from the same place where you keep your other artifacts. See directly from Google Drive when the board is edited and by whom.
GitHub Integration You can upload diagrams directly from Sketchboard.io to GitHub. You can keep documentation and images on a same repository. ✔ GitHub integration works only on public repositories at the moment, read more
Flowdock Integration Keep your team up to date, what is just designed. Send board snapshot image to a flow. ✔ Sketchboard.io sends automatic messages to a flow when board is shared within an account or board is commented.
